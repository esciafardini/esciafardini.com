Multicursor

Visual block workflow with multicursor.nvim:

  1. Ctrl+v to enter visual block mode
  2. Select the column of text
  3. I to insert at start / A to append at end
  4. Type your edit — all cursors update simultaneously
  5. Esc to clear cursors

My Config

Config Structure

~/.config/nvim/
  init.lua                  # Entry point: loads options, lsp, lazyconfig
  lua/
    options.lua             # Settings, keymaps, autocmds
    lsp.lua                 # LSP attach keybinds + server configs
    lazyconfig.lua          # Bootstraps lazy.nvim plugin manager
    whitespace.lua          # Custom whitespace module
    plugins/
      colorscheme.lua       # Kanagawa theme (customized palette)
      completions.lua       # blink.cmp + LuaSnip + lazydev
      conjure.lua           # REPL connections (Clojure, SQL)
      csv.lua               # CSV highlighting (custom plugin)
      fzf-lua.lua           # Fuzzy finder
      git.lua               # LazyGit + gitsigns
      hex.lua               # Hex editor
      lualine.lua           # Statusline + buffer tabs
      multicursor.lua       # Visual block multicursor
      neotree.lua           # File explorer (float)
      notify.lua            # nvim-notify
      obsidian.lua          # Obsidian vault integration
      paredit.lua           # Structural editing (Lisp)
      surround.lua          # mini.surround + node wrapping
      treesitter.lua        # Treesitter + textobjects
      whichkey.lua          # Keybinding hints

Core Files

init.lua — Three lines. Loads options, lsp, and lazyconfig in order.

options.lua — Global settings and keymaps.

  • Leader: Space, Local Leader: ,
  • System clipboard (unnamedplus), relative line numbers, 2-space indentation
  • Smart case search, split below/right, persistent undo
  • Auto-pairs for ( [ { in all filetypes
  • Yank highlight (purple bg, green fg, 1s)
  • Fold views auto-saved/restored
  • Clojure indent patterns for HTTP route macros
  • Fennel filetype registration

lsp.lua — LSP-attach keybinds (definition, references, code actions, rename, format). Configures 6 language servers. Removes default gr* keymaps to keep things clean.

lazyconfig.lua — Bootstraps lazy.nvim from git, auto-imports everything in plugins/. Change detection disabled. Auto-checker enabled (silent).

Plugins

Appearance
kanagawa.nvim
Dark colorscheme. Custom palette: muted yellows/oranges replaced with teal tones. Custom line number highlights.
lualine.nvim
Statusline + buffer tabs. Moon-phase mode indicator. Shows LSP status and filetype icon.
nvim-notify
Pretty notification popups. Replaces vim.notify globally.
which-key.nvim
Keybinding hints popup. Groups organized by prefix. Hides unused z-fold keys.
Navigation
neo-tree.nvim
File explorer in floating window. Opens on startup if no args. Shows dotfiles, hides node_modules/.git.
fzf-lua
Fuzzy finder for files, grep, buffers, recent files. Ctrl-j/k navigation, cycling enabled.
Editing
blink.cmp
Completion engine. Tab/S-Tab/CR keymap. Sources: buffer, LSP, path, snippets, lazydev.
LSP source requires 3+ chars. No auto-popup on keyword.
LuaSnip
Snippet engine. Loads VSCode-format snippets from config snippets/ dir.
mini.surround
Surround operations (sa/ds/cs). Node-wrapping keymaps for non-Lisp filetypes via localleader.
multicursor.nvim
Visual block → multicursor. I to insert, A to append. Esc clears.
treesitter + textobjects
Syntax highlighting, af/if function textobjects, as scope selection.
Lisp / REPL
conjure
REPL client for Clojure, Fennel, Python, Lua, SQL. Manual connect only (auto-repl disabled). Shadow CLJS support with TCP probe.
nvim-paredit
Structural editing for Lisp. Wrap/raise/navigate forms. Clojure-specific log/spy and log/daff wrapping with treesitter-based cleanup.
Git
lazygit.nvim
Full LazyGit TUI inside Neovim.
gitsigns.nvim
Git hunk signs in gutter. Stage/unstage/reset hunks. Inline preview. Blame line/buffer. Custom fork with staged sign support.
Red = unstaged, Green = staged. Obsidian vault buffers excluded.
Other
obsidian.nvim
Obsidian vault integration. Quick switch, search, daily notes with exercise templates. Custom checkbox rendering.
csv_hl.nvim
CSV syntax highlighting. Local plugin from ~/Projects/csv_hl.nvim.
hex.nvim
Hex editor for binary files.
lazydev.nvim
Neovim Lua development helper. Suppresses vim global warnings, provides luv library types.
